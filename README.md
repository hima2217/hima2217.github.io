# 🎾 ACE Club — Прилоэение для теннисного клуба

## 📌 О проекте

ACE Club — это современное интерактивное веб-приложение теннисного клуба, разработанное в рамках учебного проекта по UX/UI-дизайну и frontend-разработке.

Проект представляет собой адаптивную посадочную страницу (SPA), которая позволяет пользователям:
- записываться на тренировки,
- бронировать теннисные корты,
- выбирать тренеров,
- просматривать расписание и информацию о клубе,
- управлять своим профилем.

---

## 🎯 Цель проекта

Создать удобный и современный цифровой интерфейс для теннисного клуба, который:
- упрощает процесс записи на тренировки,
- минимизирует количество действий пользователя,
- обеспечивает быстрый доступ к ключевым функциям,
- корректно работает на мобильных и десктопных устройствах.

---

## 👥 Целевая аудитория

- Студенты 18–25 лет  
- Работающие специалисты 25–40 лет  
- Любители тенниса и постоянные клиенты клубов  

Пользователи ожидают:
- быстрый интерфейс,
- простую навигацию,
- минимальное количество шагов до записи.

---

## ⚙️ Функциональность

### Основные возможности:
- 📅 Расписание тренировок с фильтрацией по времени  
- 🎾 Бронирование кортов  
- 👨‍🏫 Выбор тренеров  
- 👤 Профиль пользователя  
- ⚡ Мгновенная запись через модальное окно  

### Интерактивные элементы:
- модальные окна записи
- анимации появления контента
- hover-эффекты карточек
- toast-уведомления
- плавная навигация по секциям

---

## 🧠 UX-особенности

- Mobile-first подход
- Минимум шагов до целевого действия (3 клика до записи)
- Чёткая визуальная иерархия
- Интерактивная обратная связь
- Адаптивность под все устройства

## 📊 UX-логика

Пользовательский сценарий:

1. Открывает главную страницу

4. Выбирает время и корт
